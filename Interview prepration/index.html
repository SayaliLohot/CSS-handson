CSS Interview Prepration Day 1

Q1. Explain inline, internal and external stylesheet

  • These three are ways of inserting a style sheet.
  • External CSS :
    • External styles are defined within the < link > element, inside the <head> section of an HTML page
    • <link rel="stylesheet" href="style.css">
  • Internal CSS :
    • Internal styles are defined within the <style> element, inside the <head> section of an HTML page
    • <style>
      body {
      background-color: linen;
      }
      </style>
  • Inline CSS :
    • Inline styles are defined within the "style" attribute of the relevant element
    • <p style="color:red;">This is a paragraph./ p>

Q2. What is css box model?

  • The CSS box model is essentially a box that wraps around every HTML element.
  • It consists of: margins, borders, padding, and the actual content.

Q3. What is the difference between padding and margin?

  • Margin is said to be the outer space of an element, i.e., the margin is the space outside of the element's border.
  • Padding is said to be the inner space of an element, i.e., the padding is the space inside of the element's border.

Q4. Difference between flex row and column?

  • These are values of flex-direction property.
  • Row represents the horizontal axis oriented from left to right
  • Column: same as row but top to bottom

Q6. Difference between absolute and relative positioning

  • Relative places an element relative to its current position without changing the layout around it.(The space will remains as it is, it will not fill by next element.)
  • Absolute places an element relative to its parent's position and changing the layout around it.(It moves acording to parent element, space aquired by other elements.)

Q7. What is the z-index in CSS?

  • The z-index CSS property sets the z-order of a positioned element
  • It defines the order of overlapping HTML elements. Elements with a higher index will be placed on top of elements with a lower index.
  • It does not work on static position.
