Q1.What is box sizing?

  • Defines how the width and height of an element are calculated: should they include padding and borders, or not.
    1. Border-box:The size of border-box remains same while size of content decreases as padding and border grows.
    2. Content-box:The content size remains same while border-box size grows as padding and border grows.

Q2.What is animation delay?

  • The animation-delay CSS property specifies the amount of time wait to perform the animation.
  • The animation-delay value is defined in seconds (s) or milliseconds (ms).

Q4.What is grid-template-columns used for?

  • The grid-template-columns property specifies the number (and the widths) of columns in a grid layout.
  • We can code it like "grid-template-columns: 30px 200px auto 100px;"

Q5.Which property you will use to merge cells vertically in grid?

  • The grid-column property specifies a grid item's size and location in a grid layout, and is a shorthand property for the following properties:- grid-column-start: and grid-column-end:
  • By using grid-column property we can achieve the merging of columns.

Q6.Which property you will use to merge cells horizontally in grid?

  • The grid-row property specifies a grid item's size and location in a grid layout, and is a shorthand property for the following properties:- grid-row-start: and grid-row-end:
  • By using grid-row property we can achieve the merging of rows.

Q7.What is the difference between display none and visibility hidden?

  • Display:none- It removes the element from the webpage layout completely,without leaving any space.( It removes the element from the document hence it does not take up any space.)
  • Visibility: hidden- It keeps the element in the webpage layout but hides it, thus leaving a blank space for the element.(It hide element with space)
